## Quick Start

1. Clone this repo
2. Set environment variables (see below)
3. Deploy to Railway

## Environment Variables

```
BOT_TOKEN=your_telegram_bot_token
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_anon_key
ADMIN_ID=your_telegram_user_id
```
